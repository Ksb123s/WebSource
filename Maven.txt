1.maven 설치
    - 자바 빌드 도구 : 빌드 자동화
    - 빌드 : 프로젝트 생성 => 라이브러리 설정=> 코드 작업 => 컴파일 => test => 패키지 생성 => 배포
    - 패키지 : 압축파일(jar, war)
    - 명령어 : mvn compile/ mvn package
